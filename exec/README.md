## Fashionity

패션 커뮤니티와 컨설팅을 동시에

<br/>

## 🚀 Services

1. 패션 커뮤니티를 통해 자신의 패션을 자랑하고 타인의 패션 참고 가능
2. 1대1 화상 컨설팅으로 맞춤형 패션 컨설팅 가능
